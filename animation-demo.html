使用指南

CSS 动画

<!-- 基础动画 -->
<div class="animate-fade-in">内容</div>
<div class="animate-slide-up">内容</div>
<div class="animate-scale-in">内容</div>

<!-- 序列动画 -->
<div class="animate-stagger">
  <div>项目 1</div>
  <div>项目 2</div>
  <div>项目 3</div>
</div>

<!-- 过渡效果 -->
<button class="transition-all btn-hover-lift">
  悬停我
</button>

Framer Motion

import { motion } from 'framer-motion';
import { fadeVariants, staggerContainer } from './framer-motion';

// 基础动画
<motion.div
  initial="hidden"
  animate="visible"
  exit="exit"
  variants={fadeVariants}
>
  内容
</motion.div>

// 序列动画
<motion.div
  initial="hidden"
  animate="visible"
  variants={staggerContainer}
>
  {items.map((item, i) => (
    <motion.div key={i} variants={staggerItem}>
      {item}
    </motion.div>
  ))}
</motion.div>

性能提示

  • 只动画 transform 和 opacity
  • 使用 will-change 优化，但及时移除
  • 避免动画布局属性（width, height, margin）
  • 尊重 prefers-reduced-motion
