01 / ARCHITECTURE

Five-Layer Architecture

From kernel-level safety to autonomous evolution, each layer builds on the last.

🛡

Safety Layer

AST-level bash guard (42 DANGER signals), mutation testing gate with 6 strategies, tool guard with circuit breaker pattern. Safety enforced at kernel level, not prompt level.

👁

Perception

Seven-sense metacognition with OpenTelemetry-native tracing. O2 vitality, Nociception pain, Chronos timing, all quantified and queryable.

🧠

Knowledge

FTS5 + vector + graph hybrid retrieval with RRF fusion. Auto-capture memory with Ebbinghaus decay across 4 memory types.

⚡

Execution Layer

Crash-proof task planning with 6 lifecycle hooks and SHA-256 attestation. Git-native coordination protocol (GNAP). Seven PostToolUse hooks for automated quality gates.

🌱

Evolution

DSPy-inspired rule health scoring. Self-evolving framework pipeline: raw to observed to validated to hardened. Reflection engine with structured feedback loops.

02 / CAPABILITIES

Core Systems

Each module is independently testable, collectively forming the harness.

🛡

Intelligent Recovery

6 error categories, 9 recovery strategies, circuit breaker pattern. Absorbed from MUNDO v2.2.3.

📊

Cost Tracker

7 waste categories, A-F health grade, per-rule token attribution. CodeBurn-inspired with percentage-based scoring.

🧬

Mutation Gate

6 Python mutation strategies. MUTATION VERIFY step in TDD pipeline. SWE-Mutation (ACL 2026) validated.

📋

Task Planning

Filesystem as external working memory. Deterministic completion gate judges disk artifacts, not transcripts.

🎨

Design Knowledge

67 UI styles, 161 color palettes, industry-specific anti-patterns. Absorbed from UI UX Pro Max (92K stars).

📡

Observability

7-sense radar charts, SQLite trace storage, Langfuse-compatible JSON export. Zero external dependencies.

CLI Tools

python3 phoenix-cost-tracker.py report Token health report with A-F grade and waste analysis

python3 phoenix-plan.py init --slug "task" Crash-proof task planning with SHA-256 attestation
python3 knowledge-base.py search "q" --hybrid FTS5 + vector + graph hybrid retrieval with RRF fusion
python3 phoenix-observability.py dashboard 7-sense radar chart with Langfuse-compatible traces
