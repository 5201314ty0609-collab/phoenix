02 / 架构 · ARCHITECTURE

五层架构 Five-Layer Architecture

从内核级安全到自主进化，每一层都建立在上一层之上。From kernel-level safety to autonomous evolution, each layer builds on the last.

🛡

安全层 Safety Layer

AST 级 Bash 安全门禁（42 个危险信号）、6 策略变异测试门控、熔断器模式工具防护。安全在内核级强制执行，而非提示词级。AST-level bash guard (42 DANGER signals), mutation testing gate with 6 strategies, tool guard with circuit breaker pattern.

👁

感知层 Perception

八感元认知与 OpenTelemetry 原生追踪。O2 生命力、痛觉、时间、空间、平衡、回声、漂移、本体——全部可量化、可查询。8-sense metacognition with OTel-native tracing. All quantified and queryable.

🧠

知识层 Knowledge

FTS5 + 向量 + 图混合检索与 RRF 融合。自动捕获记忆，艾宾浩斯衰减评分，覆盖 4 种记忆类型。FTS5 + vector + graph hybrid retrieval with RRF fusion. Auto-capture memory with Ebbinghaus decay.

⚡

执行层 Execution Layer

崩溃安全的任务规划（6 个生命周期钩子 + SHA-256 认证）。Git 原生协调协议（GNAP）。7 个 PostToolUse 钩子实现自动化质量门控。Crash-proof task planning with 6 lifecycle hooks and SHA-256 attestation. Git-native coordination protocol.

🌱

进化层 Evolution

DSPy 启发的规则健康评分。自进化框架管道：原始→观察→验证→硬化。反思引擎与结构化反馈循环。DSPy-inspired rule health scoring. Self-evolving framework pipeline: raw→observed→validated→hardened.
