🚀 部署你自己的 PHOENIX

Deploy Your Own Instance
# 1. 克隆仓库
git clone https://github.com/5201314ty0609-collab/phoenix.git
cd phoenix

# 2. 创建你的身份（只需一次）
python3 user_manager.py setup

# 3. 启动控制台
python3 server.py

# 4. 打开浏览器
http://127.0.0.1:8765

零依赖。Python 3.9+ 即可。不需要 pip install 任何东西。
